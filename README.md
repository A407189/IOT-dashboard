# 🌐 IoT Dashboard with Flask

## 📌 Project Overview
This is a simple IoT Dashboard web application built using Flask (Python).  
It includes user authentication (login system) and a real-time dashboard that displays simulated IoT sensor data like temperature and voltage.

The dashboard updates automatically and shows live charts using Chart.js.

---

## 🚀 Features
- 🔐 Secure Login System (Username & Password)
- 📊 Real-time IoT Dashboard
- 🌡️ Temperature monitoring
- ⚡ Voltage monitoring
- 📈 Live updating graph using Chart.js
- ⚠️ Alert system for high temperature
- 🔄 Auto-refresh every 2 seconds

---

## 🛠️ Tech Stack
- Python
- Flask
- HTML5
- CSS3
- JavaScript
- Chart.js
